Project: JARVIS-Multimodal-AI

Field: Artificial Intelligence / Inference Optimization

1. Abstract
This research paper details the architecture and deployment of JARVIS, an advanced multimodal AI assistant engineered for high-performance reasoning on resource-constrained edge hardware. By orchestrating a pipeline between Microsoft’s Phi-3-mini (3.8B parameters) and OpenAI’s Whisper-tiny, this project demonstrates that low-latency, complex AI interactions are achievable without high-end GPU dependency.

3. Methodology & Technical Architecture
The JARVIS engine is built on a "Lean-Inference" philosophy:

Acoustic Processing: Leveraging whisper-tiny for sub-500ms speech-to-text transcription.

Neural Reasoning: Integration of Phi-3-mini-4k-instruct using optimized transformers pipelines to handle complex logical queries.

Full-Stack Deployment: Hosted via a Gradio-based interface on Hugging Face Spaces, achieving near-instantaneous end-to-end response cycles.
